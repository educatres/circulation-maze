# 血球迷宮：循環運輸大挑戰

一套適合國中生物課程的循環系統互動教材。玩家化身紅血球，在迷宮中完成肺部氣體交換、小腸養分吸收、肌肉供能、肝腎廢物處理與腦部供應等任務。

## 教學重點

- 認識心臟、肺、小腸、肝臟、腎臟、腦與肌肉在循環系統中的角色。
- 判斷氧氣、二氧化碳、葡萄糖、胺基酸、尿素、水與鹽類的運輸方向。
- 理解微血管是血液與組織細胞進行物質交換的主要場所。
- 透過迷宮、知識閘門、即時回饋與計分，強化循環路徑概念。

## 遊戲特色

- 5 個循序漸進的關卡。
- 方向鍵、WASD 與行動裝置按鈕操作。
- 題目直接融入迷宮，答對才能通過知識閘門。
- 分數、生命值、連續答對與學習紀錄。
- RWD 響應式版面，適合電腦、平板與手機。
- 無需後端、無需資料庫，可直接部署在 GitHub Pages。

## 本機執行

直接開啟 `index.html`，或使用靜態伺服器：

```bash
python -m http.server 8000
```

然後開啟 `http://localhost:8000`。

## 教師使用建議

- 課前：讓學生先閱讀任務手冊。
- 課中：兩人一組，一人操作、一人說明選路理由。
- 課後：請學生畫出一條完整路徑，例如「小腸 → 肝臟 → 心臟 → 肌肉」，並標示運輸物質。
